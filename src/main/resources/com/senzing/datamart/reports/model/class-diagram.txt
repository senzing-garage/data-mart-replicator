FUNCTIONAL GROUPS
═════════════════

The model is organized into 4 main functional groups:

1. Loaded Stats - Basic record/entity counts per data source
   (SzLoadedStats → SzSourceLoadedStats)

2. Summary Stats - Cross-source matching statistics with match/relation counts
   broken down by match key and principle
   (SzSummaryStats → SzSourceSummary → SzCrossSourceSummary → SzMatchCounts / SzRelationCounts)

3. Breakdowns - Distribution histograms for entity sizes and relation counts
   (SzEntitySizeBreakdown → SzEntitySizeCount, SzEntityRelationsBreakdown → SzEntityRelationsCount)

4. Paging/Detail - Paginated entity and relation results with full detail
   (SzEntitiesPage → SzReportEntity → SzReportRecord, SzRelationsPage → SzReportRelation)
